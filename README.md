# Polish Payments API
PHP SDK for easier integration with polish payment providers.

## Documentation
Documentation can be found here: https://patryqhyper.gitbook.io/polish-payments-api-v4/

## Installation
Installation via Composer:
```
composer require patryqhyper/polish_payments_api:^4.0
```

## Supported providers:
- CashBill.pl
- Dpay.pl
